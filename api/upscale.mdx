The `/v1/upscale` endpoint uses AI super-resolution to enlarge an image while preserving and recovering detail. Submit your image by URL or as a base64 string, and receive a link to the upscaled result.

## Endpoint

```
POST https://api.artnex.ai/v1/upscale
```

## Request parameters

<ParamField body="image" type="string" required>
  The image to upscale. Provide either:
  - A publicly accessible HTTPS URL pointing to the image, or
  - A base64-encoded image string with a data URI prefix (e.g., `data:image/jpeg;base64,...`).

  Supported formats: JPEG, PNG, WebP. Maximum input file size: 10 MB.
</ParamField>

<ParamField body="scale" type="number" default="2">
  The upscale multiplier applied to the original dimensions. For example, a `2` on a 512×512 image produces a 1024×1024 output. Common values: `2`, `4`.
</ParamField>

## Response fields

<ResponseField name="success" type="boolean" required>
  `true` when the request completed successfully.
</ResponseField>

<ResponseField name="image_url" type="string" required>
  Direct HTTPS URL to the upscaled image. Available for 24 hours; download promptly if you need to keep the result.
</ResponseField>

<ResponseField name="request_id" type="string">
  Unique identifier for this request.
</ResponseField>

<ResponseField name="credits_used" type="integer">
  Credits deducted from your balance. Upscaling costs **100 credits** per request.
</ResponseField>

## Credit cost

| Operation | Credits |
|---|---|
| Upscale (any scale factor) | 100 |

## Example requests

<CodeGroup>

```bash curl
curl --request POST \
  --url https://api.artnex.ai/v1/upscale \
  --header "Authorization: Bearer YOUR_API_KEY" \
  --header "Content-Type: application/json" \
  --data '{
    "image": "https://example.com/product-photo.jpg",
    "scale": 4
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.artnex.ai/v1/upscale",
    headers={
        "Authorization": "Bearer YOUR_API_KEY",
        "Content-Type": "application/json",
    },
    json={
        "image": "https://example.com/product-photo.jpg",
        "scale": 4,
    },
)

data = response.json()
print("Upscaled image URL:", data["image_url"])
print("Credits used:", data["credits_used"])
```

```javascript JavaScript
const response = await fetch("https://api.artnex.ai/v1/upscale", {
  method: "POST",
  headers: {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    image: "https://example.com/product-photo.jpg",
    scale: 4,
  }),
});

const data = await response.json();
console.log("Upscaled image URL:", data.image_url);
console.log("Credits used:", data.credits_used);
```

</CodeGroup>

## Error codes

| HTTP status | Error code | Cause |
|-------------|------------|-------|
| `400` | `missing_image` | The `image` field is absent |
| `400` | `invalid_image_format` | The image format is not supported or the file is corrupt |
| `400` | `image_too_large` | The image exceeds the 10 MB size limit |
| `400` | `invalid_scale` | The `scale` value is outside the supported range |
| `401` | `invalid_api_key` | The API key is missing, malformed, or revoked |
| `402` | `insufficient_credits` | Your account does not have enough credits |
| `429` | `rate_limit_exceeded` | You have exceeded your plan's daily request quota |

<Tip>
  Upscaling works best on images that are already sharp. Heavily compressed or blurry source images will produce limited improvement.
</Tip>
