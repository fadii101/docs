The `/v1/generate` endpoint converts a text prompt into one or more images using your chosen AI model. Send a prompt and model name; receive a list of image URLs you can use immediately or store for later.

## Endpoint

```
POST https://api.artnex.ai/v1/generate
```

## Request parameters

<ParamField body="prompt" type="string" required>
  The text description of the image you want to generate. Be specific — more detail leads to better results. Maximum 4,000 characters.
</ParamField>

<ParamField body="model" type="string" required>
  The AI model to use for generation. Must be one of the supported model IDs listed in the [Models](#models) section below.
</ParamField>

<ParamField body="width" type="integer">
  Image width in pixels. Accepted values depend on the model. Defaults to `1024` for most models.
</ParamField>

<ParamField body="height" type="integer">
  Image height in pixels. Accepted values depend on the model. Defaults to `1024` for most models.
</ParamField>

<ParamField body="num_images" type="integer" default="1">
  Number of images to generate in a single request. Accepted range: `1`–`4`. Each image costs the full per-model credit amount.
</ParamField>

## Response fields

<ResponseField name="success" type="boolean" required>
  `true` when the request completed successfully.
</ResponseField>

<ResponseField name="images" type="array">
  Array of generated image objects. Each object contains a `url` field with a CDN-hosted link to the image.

  <Expandable title="Image object properties">
    <ResponseField name="url" type="string">
      Direct HTTPS URL to the generated image. URLs are available for 24 hours; download and store any images you need to keep permanently.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="request_id" type="string">
  Unique identifier for this request, useful for support queries or logging.
</ResponseField>

<ResponseField name="credits_used" type="integer">
  Total credits deducted from your account balance for this request.
</ResponseField>

## Models

Choose the model that fits your quality and budget requirements:

| Model ID | Credits per image | Description |
|---|---|---|
| `z-image-turbo` | 100 | Fast, affordable generation for prototyping |
| `artsyn-light` | 240 | Balanced quality and speed |
| `flux-2-pro` | 250 | High-quality professional output |
| `seedream-v4` | 200 | Bytedance Seedream v4 |
| `seedream-v4.5` | 800 | Higher-fidelity Seedream output |
| `nano-banana-pro` | 800 | Premium generation with fine detail |
| `nano-banana-pro-ultra` | 1,000 | Ultra-high-resolution (up to 8K) output |
| `gpt-image-2` | Varies | OpenAI GPT-Image-2; credits vary by resolution and quality |

<Note>
  `gpt-image-2` uses dynamic pricing based on resolution (`1k`, `2k`, `4k`) and quality (`low`, `medium`, `high`). Pass `resolution` and `quality` fields alongside the standard parameters when using this model.
</Note>

## Example requests

<CodeGroup>

```bash curl
curl --request POST \
  --url https://api.artnex.ai/v1/generate \
  --header "Authorization: Bearer YOUR_API_KEY" \
  --header "Content-Type: application/json" \
  --data '{
    "prompt": "A serene mountain landscape at sunset, photorealistic, golden hour lighting",
    "model": "flux-2-pro",
    "width": 1024,
    "height": 1024,
    "num_images": 1
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.artnex.ai/v1/generate",
    headers={
        "Authorization": "Bearer YOUR_API_KEY",
        "Content-Type": "application/json",
    },
    json={
        "prompt": "A serene mountain landscape at sunset, photorealistic, golden hour lighting",
        "model": "flux-2-pro",
        "width": 1024,
        "height": 1024,
        "num_images": 1,
    },
)

data = response.json()
for img in data["images"]:
    print(img["url"])
```

```javascript JavaScript
const response = await fetch("https://api.artnex.ai/v1/generate", {
  method: "POST",
  headers: {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    prompt: "A serene mountain landscape at sunset, photorealistic, golden hour lighting",
    model: "flux-2-pro",
    width: 1024,
    height: 1024,
    num_images: 1,
  }),
});

const data = await response.json();
data.images.forEach((img) => console.log(img.url));
```

## Error codes

| HTTP status | Error code | Cause |
|-------------|------------|-------|
| `400` | `missing_prompt` | The `prompt` field is absent or empty |
| `400` | `invalid_model` | The `model` value is not a supported model ID |
| `400` | `invalid_dimensions` | `width` or `height` is outside the allowed range for the selected model |
| `401` | `invalid_api_key` | The API key is missing, malformed, or revoked |
| `402` | `insufficient_credits` | Your account does not have enough credits to complete the request |
| `429` | `rate_limit_exceeded` | You have exceeded your plan's daily request quota |

<Tip>
  Use `z-image-turbo` (100 credits) during development to conserve credits, then switch to a higher-fidelity model for production.
</Tip>
