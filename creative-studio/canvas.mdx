The Artnex Canvas is a full-featured graphic design editor built directly into the platform. You can draw shapes, place text, import images, remove backgrounds with one click, and ask the built-in AI assistant to generate or edit artwork ŌĆö without leaving the editor.

## Navigate to Canvas

Go to **Tools ŌåÆ Canvas** in the left sidebar. The Canvas Projects page opens, showing all your saved designs. Click **New Project** to start a fresh canvas, or click any existing project thumbnail to reopen it.

<Note>
You must be signed in to create and save canvas projects. Projects auto-save every two seconds after any change.
</Note>

## Working with layers

Click the **Layers** button in the top-right area, or press `L`, to open the Layer panel. The panel lists every object on the canvas from top (front) to bottom (back).

From the Layer panel you can:

- Toggle visibility of any layer (click the eye icon)
- Lock a layer to prevent accidental edits
- Reorder layers by dragging them up or down
- Identify object types at a glance (image, text, shape, etc.)

---

## AI Chat panel

Click the **AI** button (sparkle icon) to open the AI Chat panel. You can:

- Generate a new image by typing a description into the chat
- Send an image already on the canvas to the AI and ask it to edit or transform the image
- Switch between **Image** and **Video** generation modes using the toggle at the top of the panel

Generated results appear directly on the canvas as new image objects that you can reposition and resize.

---

## Background remover

Select any image on the canvas to reveal the object toolbar above it. Click **Remove Background** to instantly strip the background from the image. The processed result replaces the original image in place.

<Note>
Background removal requires a credit deduction. The exact cost is shown in the toolbar before you confirm.
</Note>

## Export your design

Click the **Download** button (top-right) or press `Ctrl` / `Ōīś` + `E` to export the canvas as a PNG file. The file is saved to your browser's default download location named `artnex-canvas-<timestamp>.png`.

<Tip>
If you have individual images on the canvas that you want to download separately, select the image and use the per-object download option in the object toolbar.
</Tip>

---

## Saving and managing projects

The canvas **auto-saves** every two seconds after any change. The save status indicator in the top bar shows **Saved**, **Saving**, or **Unsaved**.

To save manually at any time, press `Ctrl` / `Ōīś` + `S`.

Return to **Tools ŌåÆ Canvas** to see all your projects. From the Canvas Projects page you can:

- Open a project by clicking its thumbnail
- **Rename** a project from the three-dot menu
- **Duplicate** a project to branch off a copy
- **Delete** a project (this is permanent and cannot be undone)

<Warning>
Deleting a project is permanent. There is no way to recover a deleted project.
</Warning>

## Create your first design

<Steps>
  <Step title="Open Canvas Projects">
    Go to **Tools ŌåÆ Canvas** in the sidebar. The Canvas Projects page opens.
  </Step>
  <Step title="Create a new project">
    Click the **New Project** card. The editor opens with a blank canvas.
  </Step>
  <Step title="Add a background shape">
    Select the **Rectangle** tool from the floating toolbar. Click and drag on the canvas to draw a rectangle that covers the whole area. Use the shape toolbar that appears to set a fill color.
  </Step>
  <Step title="Add text">
    Press `T` to activate the Text tool. Click anywhere on the canvas to place a text box and type your content. Use the text toolbar to change the font, size, and color.
  </Step>
  <Step title="Import an image">
    Click the **Image** icon in the floating toolbar to open a file picker. Select an image from your computer. The image is placed on the canvas as a new layer.
  </Step>
  <Step title="Remove the image background">
    With the image selected, click **Remove Background** in the object toolbar. Artnex processes the image and replaces it with a version that has a transparent background.
  </Step>
  <Step title="Ask the AI to generate additional artwork">
    Click the **AI** button to open the AI Chat panel. Type a description of what you want, for example `A minimalist mountain landscape at sunset`, and press Enter. The generated image appears on the canvas.
  </Step>
  <Step title="Export your design">
    Press `Ctrl` / `Ōīś` + `E` to download the finished design as a PNG.
  </Step>
</Steps>
